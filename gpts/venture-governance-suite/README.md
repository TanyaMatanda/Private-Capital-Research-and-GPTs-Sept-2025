# Venture Governance GPT Suite

Purpose
-------
A small suite of prompt templates and artifacts to help venture teams and LP governance bodies operationalize governance for venture portfolios. Use cases:
- Generate board/committee briefing notes summarizing governance health.
- Produce diligence question sets for early-stage governance reviews.
- Suggest standard operating procedures for recurring governance tasks.

Intended users
--------------
- VC operating partners
- LP governance and monitoring teams
- Startup board members and counsel (for non-legal suggestions)

Inputs
------
- One-page governance summaries, portfolio company updates, board minutes excerpts.

Outputs
-------
- 1-page governance health summary
- Tailored board briefing bullets
- Governance checklist for portfolio company maturity stage
